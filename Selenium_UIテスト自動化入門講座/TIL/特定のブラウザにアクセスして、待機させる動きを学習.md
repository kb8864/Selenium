## 学習内容
・自動化テストを行うには、まず「誰が（Driver）」「どこへ（URL）」行くかを指定する必要がある
```
WebDriver driver = new ChromeDriver();//この部分をIEや、Firefoxなどブラウザを指定できる
driver.get("https://www.google.com/");

```
・driver.quit()をつけた理由

テストが終わったらブラウザを閉じ、PCのメモリを解放するため
